# 知识便利贴 · 计算器示例

本文对应示例：**Python → 计算器**。下面梳理代码里出现的几个 Python 知识点；页面右侧「知识便利贴」只展示每条的一句话摘要。

## 1. 类（class）

用 `class Calculator` 把「屏幕显示、累计值、运算符」等状态，以及输入数字、运算等方法收在一起。创建实例后，状态跟着对象走，比一堆散落的全局变量更好维护。

## 2. math 库

`import math` 后可使用 `math.sin`、`math.cos`、`math.log`、`math.pi` 等。注意：标准库按**弧度**算三角函数，本示例在调用前用 `math.radians` 把「度」转成弧度；绝对值请用内置 `abs`，没有 `math.abs`（若要浮点绝对值可用 `math.fabs`）。

## 3. lambda 表达式

`lambda x: x * x` 是匿名的小型函数，适合只在一处、逻辑很短时传入。科学计算里多处 `_unary(..., lambda x: ...)` 就是把「怎么算」临时传给公共方法。

## 4. 高阶函数

高阶函数指「接收函数作参数」或「返回函数」的函数。本例中 `_unary(self, _name, fn)` 接收 `fn`，再执行 `fn(value)`，因此 `_unary` 是高阶函数；传入的可以是 `math.sqrt`，也可以是 `lambda`。

## 5. @staticmethod 装饰器

`@staticmethod` 标在 `_fmt` 上，表示它不依赖实例状态（不用 `self`），只根据传入的数值格式化字符串。调用时仍可写 `self._fmt(n)`，本质是挂在类上的普通函数。

## 6. 内置函数（如 abs）

`abs` 属于 Python **内置函数**，无需 import，直接写 `abs(-7)`。它不在 `math` 模块里，因此不能写 `math.abs`。

## 友情提示

类里定义的函数，一般称为**方法（method）**；但若某个方法接收或返回函数（如本例的 `_unary`），从概念上它仍然是**高阶函数**——「方法」说的是它属于类，「高阶」说的是它的用法特征，两者可以同时成立。
