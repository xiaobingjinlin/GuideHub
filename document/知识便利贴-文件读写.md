# 知识便利贴 · 文件读写

本文对应示例：**Python → 文件读写**。右侧「知识便利贴」只展示每条的一句话摘要；下文为展开说明。

---

## 1. Path(__file__).resolve().parent

`__file__` 是当前 `.py` 文件路径；`.resolve()` 得到绝对路径；`.parent` 取其父目录。  
再用 `/` 拼接文件名，例如：

```python
FILE_PATH = Path(__file__).resolve().parent / "memo.txt"
```

这样 `memo.txt` 会落在与本模块同目录，路径稳定、不依赖启动时的工作目录。

## 2. with open(...)

`with open(...) as f:` 会在进入时打开文件，离开代码块时**自动关闭**文件流（即使中途出错也会关），不必手写 `f.close()`。

## 3. 常用打开模式

| 模式 | 含义 |
| --- | --- |
| `r` | 读（文件需已存在） |
| `w` | 写（覆盖；不存在则创建） |
| `a` | 追加到末尾 |

本示例读用 `r`，整份覆写/清空用 `w`。

## 4. encoding="utf-8"

文本读写建议写上 `encoding="utf-8"`，避免不同系统默认编码不一致导致中文乱码。
