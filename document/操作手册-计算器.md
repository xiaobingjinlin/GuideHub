# 计算器操作手册

本手册对应示例：**Python → 计算器**。点击「运行」后，可用顶部按钮在 **简单计算器** 与 **科学计算器** 之间切换。

## 通用说明

- 三角函数按 **角度制（度）** 计算，不是弧度。
- 数字键、小数点不会写入后台控制台；只有真正算出结果时才输出（如 `30 + 5 = 35`、`sin(30) = 0.5`）。
- `C` 清空全部状态；科学模式下的 `±` 切换正负号。

## 简单计算器

| 操作 | 按键顺序（示意） | 预计结果 |
| --- | --- | --- |
| 加减 | `12` `+` `3` `=` | `15` |
| 乘除 | `10` `÷` `4` `=` | `2.5` |
| 小数 | `1` `.` `5` `×` `2` `=` | `3` |

## 科学计算器

切换到科学模式后会出现 `sin / cos / tan / ln / log / √ / x² / xʸ / π / e / ± / 1/x / |x|` 等按键。

| 表达式（含义） | 操作步骤 | 预计结果 |
| --- | --- | --- |
| \(\sin 30^\circ\) | 输入 `30` → `sin` | `0.5` |
| \(\cos 60^\circ\) | 输入 `60` → `cos` | `0.5` |
| \(\tan 45^\circ\) | 输入 `45` → `tan` | `1` |
| \(\sqrt{9}\) | 输入 `9` → `√` | `3` |
| \(5^2\) | 输入 `5` → `x²` | `25` |
| \(2^8\) | 输入 `2` → `xʸ` → `8` → `=` | `256` |
| \(\ln e\) | 按 `e` → `ln` | `1` |
| \(\log_{10} 100\) | 输入 `100` → `log` | `2` |
| \(\pi\) | 按 `π` | `3.14159265359`（约） |
| \(1/4\) | 输入 `4` → `1/x` | `0.25` |
| \(\lvert -7 \rvert\) | 输入 `7` → `±` → `|x|` | `7` |

若结果与上表偏差极大，请确认当前是科学模式，且三角函数输入的是**角度**而不是弧度。
