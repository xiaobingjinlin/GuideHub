# 知识便利贴 · 力扣算法-分发糖果

对应题目：[LeetCode 135. 分发糖果](https://leetcode.cn/problems/candy/)  
页面菜单：**Python → 力扣算法-分发糖果**；右侧「知识便利贴」只展示每条的一句话摘要。

## 1. 列表乘法初始化

```python
candies = [1] * n
```

生成长度为 `n`、每个元素都是 `1` 的列表，对应题意「每人至少一颗糖」。注意：若元素是可变对象（如列表），`*` 复制的是引用，这里用不可变的 `1` 没有问题。

## 2. range(1, n)：正向扫描下标

```python
for i in range(1, n):
```

`range` 是**左闭右开**区间：`range(start, stop)` 取到 `start … stop-1`，默认 `step=1`。  
因此 `range(1, n)` 的取值是 `1, 2, …, n-1`，正好覆盖数组除第一个外的所有下标，适合「和左邻居比较」。

## 3. range(n - 2, -1, -1)：反向扫描下标

```python
for i in range(n - 2, -1, -1):
```

同样是左闭右开：从 `n-2` 走到 `-1`（不含），步长 `-1`，实际取值是 `n-2, n-3, …, 0`。  
从倒数第二个下标扫到 `0`，适合「和右邻居比较」。

## 4. 内置函数 sum

```python
return sum(candies)
```

`sum` 是内置函数，对可迭代对象中的数值求和，这里直接得到最少糖果总数。
