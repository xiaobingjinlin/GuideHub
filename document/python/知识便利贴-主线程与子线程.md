# 知识便利贴 · 主线程与子线程

本文对应示例：**Python → 主线程与子线程**。右侧「知识便利贴」只展示每条的一句话摘要；下文为展开说明。

---

## 1. threading.Thread 创建子线程

`threading.Thread(target=..., daemon=...)` 创建一个线程实例：`target` 指定子线程要跑的函数；创建后还要 `start()` 才会真正开始执行。

## 2. daemon=True 粗暴回收

`daemon=True` 表示守护线程：主程序（主线程）退出时，这类子线程会被直接回收，不保证跑完、也不保证优雅收尾。适合「可丢弃」的后台逻辑。

## 3. t.join() 等待子线程结束

`t.join()` 让**当前线程**（通常是主线程）停下来，等到子线程 `t` 跑完再继续。演示里用它是为了等任务做完再比总耗时；它本身不会取消 `daemon`。

## 4. 何时不必 join

真实业务里若主线程后续工作远长于子线程，子线程往往在主线程还在忙时就已结束，很多场景不必再写 `join()`；若主程序即将退出且线程是 daemon，通常也不 join，而是接受可能被粗暴打断。
